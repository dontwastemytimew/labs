Detailed Description

Дозволяє швидко генерувати випадкові значення для графів та транспортних об'єктів.

Шаблони та спеціалізації для числових типів, рядків, векторів та об'єктів BaseVehicle.

template<typename T>
T generateRandomEdge ( const T & maxValue)
inline

Загальний шаблон для генерації випадкового значення типу T.

Для арифметичних типів повертає випадкове число від 0 до maxValue. Для інших типів повертає "пустий" об'єкт T.
